# nvchad-config

This is my personal neovim configuration I use on my Raspberry Pi. It is based on [nvchad](https://nvchad.com/)'s example configuration.

## Installation
### Prerequisites
> - [nvchad](https://nvchad.com/)

Simply clone this repository into your `~/.config/nvim/lua/custom` directory and run `:LazyInstall` in neovim to install the missing plugins.

```bash
git clone https://github.com/guezey/nvchad-config.git ~/.config/nvim/lua/custom
```
